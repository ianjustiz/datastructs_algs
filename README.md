# datastructs_algs
Implementation of data structures and algorithms in C.

Current functionality:
  - Linked List w/ Tail Pointer
  - Stacks
  - Queues
  - Sorting Algorithms
    - Quicksort
    - Merge Sort
    - Radix Sort
    - Bucket Sort
    - Insertion Sort
    - Selection Sort
    - Bubble Sort
  - Binary Search Trees
  - AVL Trees (WIP - Experimental)
